**Product Name:** GraQle

**Category:** Developer Tools / Code Analysis / AI Tools
**Website:** https://graqle.com

**Short Description (150 chars):**
Knowledge graph engine for codebases. Gives AI tools architecture-aware reasoning. MCP server for Claude, Cursor, Copilot.

**Medium Description (300 chars):**
GraQle builds a knowledge graph from any codebase so AI coding tools reason over structure, not strings. Dependency analysis, impact analysis, institutional memory. 16-tool MCP server for Claude Code, Cursor, VS Code Copilot. 14 LLM backends. Fully offline capable. pip install graqle.

**Long Description (500 chars):**
GraQle is a Python SDK + CLI that builds knowledge graphs from codebases, giving AI coding tools architecture-aware reasoning. Instead of reading 60 files (50,000 tokens) to answer "what depends on auth?", your AI queries the graph and gets a precise answer in 500 tokens. Features include dependency analysis, impact analysis, institutional memory, safety checks, and a 16-tool MCP server for Claude Code, Cursor, and VS Code Copilot. Supports 14 LLM backends including fully offline options with Ollama.
