# ADR-105: Streaming Intelligence & Quality Gate

**Date:** 2026-03-15 | **Status:** ACCEPTED

**Context:** AI coding tools bypass Graqle MCP tools 70-80% of the time in favor of grep/read/bash. Current batch scan takes 2-5 minutes with no intermediate value. Validation is post-hoc, not inline.

**Decision:** 3-layer quality gate (embedded intelligence + deep reasoning + enforcement) with streaming per-file validation pipeline delivering first value in <3 seconds.

**Consequences:** Transforms Graqle from passive tool (70-80% bypass) to embedded intelligence layer (1-2% bypass). Requires new intelligence/ module but is fully additive — zero changes to existing code.
