\noindent\textbf{Title:} CogniGraph: Graph-of-Agents Distributed Reasoning over Knowledge Graph Topologies with Prize-Collecting Steiner Tree Activation and Convergent Message Passing

\vspace{6mm}

\noindent A computer-implemented system and method for distributed reasoning over knowledge graphs, wherein each node of a knowledge graph hosts an autonomous language model agent. For each query, a Prize-Collecting Steiner Tree (PCST) optimization selects a compact subgraph of query-relevant nodes, instantiating reasoning agents only on selected nodes. The agents conduct multiple rounds of message-passing reasoning along graph edges, with convergence detected via inter-message similarity metrics, enabling early termination. A MasterObserver module provides full transparency by monitoring all inter-agent traffic without participating in reasoning. A backend fallback chain supports heterogeneous inference backends (cloud APIs, local GPU models, adapter-augmented models) with automatic failover and per-query cost budget enforcement. Final answers are synthesized through graph-topology-aware hierarchical aggregation that preserves the knowledge graph's structural hierarchy. Benchmark evaluation on HotpotQA and EU-RegQA demonstrates improved multi-hop reasoning accuracy over single-agent baselines while maintaining transparency and cost control.
